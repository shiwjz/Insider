# INSIDER (인싸 호소인)

> 클릭으로 ‘좋아요’를 모아 친구를 해금하고,  
> 각 친구의 전용 아이템을 구매하면 스토리와 외형이 변화하는  
> **모바일 클리커 게임**입니다.

1학년 때 동아리에서 제작한 첫 Unity 프로젝트이며,

# INSIDER

> A mobile clicker game where players earn “Likes” by tapping,  
> unlock friends, and purchase each character’s unique item to reveal their story and change their appearance.

This was my first Unity project, developed in a freshman club team within **2 weeks**.

---

## 🎮 Overview
- Genre: Mobile Clicker
- Platform: Android
- Engine: Unity
- Duration: 2 weeks
- Team: 1 Artist / 2 Programmers / 1 Planner
- Role: Programmer

---

## ✨ Features

- Tap to earn Likes (currency)
- Unlock friends based on accumulated Likes
- Each character has one unique item
- Purchasing the item:
  - Unlocks character story
  - Changes character appearance
- Real-time UI updates based on progression

---

## 🧩 My Contributions

- Implemented tap-to-earn currency logic
- Real-time currency UI updates
- Built progression & friend unlock system
- Implemented per-character item purchase system
- Story display system triggered by item purchase
- Character appearance change logic
- Connected UI flow and button interactions
- Data persistence (PlayerPrefs if used)

---

## 📦 How to Run

### Option 1: APK
- Download the APK file from the repository
- Install and run on Android device

### Option 2: Unity
1. Open with Unity [e.g., 2021.3.x LTS]
2. Load the project
3. Run the starting scene

---

## 📌 Notes

- As my first Unity project, I focused on building a complete gameplay loop  
  (Tap → Earn Currency → Unlock → Visual Change) within a short timeframe.
- Designed and implemented core gameplay systems based on team planning and art assets.
